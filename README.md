# B-Tree Visualizer & Mini-DBMS 🌳

Десктопний застосунок, створений на **C#** та **Avalonia UI**, який візуалізує роботу структури даних **B-дерево** (B-Tree). Проєкт імітує роботу базової системи управління базами даних (СУБД), дозволяючи керувати ключами та даними в реальному часі.

## 🛠 Стек технологій
* **Мова:** C# (.NET)
* **UI Фреймворк:** Avalonia UI (Кросплатформна розробка)
* **Архітектура:** MVVM (Model-View-ViewModel)

## 📌 Основний функціонал
* **Керування B-Tree (CRUD):** Додавання, пошук, редагування та видалення записів із дотриманням усіх правил балансування B-дерева (злиття та розбиття вузлів).
* **Інтерактивна візуалізація:** Динамічна побудова дерева на екрані з можливістю масштабування. 
* **Кольорова індикація:** Підсвічування операцій у реальному часі:
  * 🟢 Доданий запис
  * 🔵 Знайдений запис (шлях пошуку)
  * 🔴 Видалений запис
  * 🟣 Злиття або перебудова вузлів
* **Стрес-тестування:** Вбудований механізм автоматичного заповнення бази на 10 000 записів для перевірки продуктивності алгоритму.

## 🗂 Архітектура структури даних
* `BTreeNode.cs` — реалізація логіки вузла B-дерева (робота з дочірніми елементами, перевірка на статус "листка", балансування).
* Вузли містять не лише ключі, а й прив'язані до них списки даних (`Records`), що повністю повторює принципи реальних індексів у реляційних базах даних.
